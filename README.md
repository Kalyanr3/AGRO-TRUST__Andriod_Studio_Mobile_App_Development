# 🌾 Agro Trust - Android Mobile App

🚀 A Blockchain & IoT based Agriculture Supply Chain Management System developed as part of a major project. This Android application ensures transparency, security, and traceability in the agricultural product lifecycle.

---

## 📱 Features

- 🔗 Blockchain-based secure data storage
- 🌾 Product traceability from farmer to consumer
- 📦 Real-time supply chain tracking
- 👨‍🌾 Farmer, Vendor & Consumer modules
- 🔐 Secure transactions & data sharing
- 📊 User-friendly interface with smooth navigation

---

## 🛠️ Tech Stack

- 📱 Android (Java/Kotlin)
- 🔗 Blockchain Technology
- 🌐 REST APIs
- ☁️ Cloud Integration
- 📡 IoT (Data Collection & Monitoring)

## 🚀 Getting Started

### Prerequisites
- Android Studio installed
- Android SDK configured

### Installation

1. Clone the repository: git clone https://github.com/Kalyanr3/Agro_Trust_Android.git

2. Open in Android Studio

3. Build & Run the project

## 🎯 Use Case

This application helps:

- Consumers to verify product authenticity
